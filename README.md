# Pandas Time Series Mini Project

## Project Overview
This is a mini project based on time series data analysis using Python Pandas.
The project focuses on understanding and handling date-time data and analyzing daily sales trends.

## Objective
- Learn how to work with date and time data in Pandas
- Perform basic analysis on daily sales data
- Convert date columns into datetime format
- Understand time-based data handling

## Tools & Libraries Used
- Python
- Pandas
- Matplotlib
- Google Colab

## Dataset Description
The dataset contains:
- Date column (daily frequency)
- Sales values for each day

## Key Concepts Covered
- Creating date ranges
- Datetime conversion
- Time series data handling
- Basic data analysis using Pandas
